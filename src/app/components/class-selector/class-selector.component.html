Comment l'utiliser

  • Choisissez l'équipe que vous voulez modifier avec Équipe gauche ou Équipe droite.
  • Cliquez sur une classe pour l'ajouter à l'équipe sélectionnée (max 6 par équipe).
  • Une fois choisie, une classe peut être retirée de l'équipe en faisant un clic-droit sur celle-ci.
  • Pour afficher la classe comme étant morte, faites un clic-gauche.
  • Vous pouvez renommer les équipes en cliquant sur leur nom.
  • Pour déplacer des classes déjà choisies au sein d'une équipe, vous pouvez les glisser-déposer à l'emplacement de votre choix.
  • 🔄 Réinitialiser vide les deux équipes.
  • 🔗 Copier le lien Overlay copie l'URL à coller dans Streamlabs/OBS, après avoir créé une nouvelle "Source" de type "Source du navigateur". Il faudra réduire la taille de la fenêtre pour n'avoir que le haut de la fenêtre d'afficher.
  • Chaque changement réalisé sur le site est synchronisé en temps réel avec l'overlay OBS, ce qui fait que vous avez une interface simple à prendre en main pour facilement paramétrer l'affichage.
